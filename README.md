## Description

This project is an implementation of Game of Life using Java.

## Features

- 25 × 25 board
- Glider initialized at the center
- Game of Life rules
- Console output

How to Run

1. Clone the repository.
2. Open the project in any Java IDE.
3. Ensure Maven dependencies are imported.
4. Run Main.java.

## Project Structure

- Cell 
- Board 
- GameEngine
- ConsoleRender
- Game
- Main
